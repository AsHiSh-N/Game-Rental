# Game-Rental

Developed a command line Game Rental app that allows users to rent gaming consoles, Controllers, and gaming Discs with different functionalities.
the end-users for this app were Sellers and Gamers.
Gamers and sellers can perform different operations based on their role like
1. Create and update products
2. View all products by their id
3. managing wishlists and carts
4. Place orders and view orders
